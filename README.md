# aea1meta

AppleEncryptedArchive metadata dumper.

### Building

```
make
```

### Usage

```
./aea1meta file.aea        - Dump all props as JSON
./aea1meta file.aea [prop] - Dump single prop value raw
./aea1meta -l file.aea     - Dump prop names, one per line
```
